# YouTube進捗管理アプリ 設計書

**作成日**: 2026-05-11  
**ステータス**: 承認済み

---

## 概要

YouTubeチャンネルを一人で運営するクリエイター向けの、動画制作進捗管理Webアプリ。企画から公開まで5つのステージを手動で管理する。YouTube APIは一切使用せず、データはブラウザのlocalStorageに保存する。

---

## ユーザー・利用シナリオ

- **対象**: 個人でYouTubeチャンネルを運営するクリエイター（1人）
- **利用目的**: 複数本の動画の制作ステージを一覧で把握し、公開スケジュールを管理する
- **利用環境**: PCブラウザ（Vercelでホスティング）

---

## 制作ステージ

| ステージ | 説明 |
|---------|------|
| 企画 | テーマ出し・アイデア整理 |
| 台本 | 台本・原稿の執筆・修正 |
| 編集 | 動画編集作業 |
| サムネイル | サムネイル画像の制作 |
| 公開 | YouTubeへのアップロード・公開済み |

## 画面構成

### ① ダッシュボード（メイン画面 `/`）

**上部サマリーエリア**
- 各ステージの動画本数をカード形式で横並び表示（企画・台本・編集・サムネイル・公開済み）
- ステージごとに色分け

**フィルター・検索バー**
- タイトルのキーワード検索
- ステージによるフィルタリング

**動画リスト**
- 各動画を1行カード形式で表示
- 表示項目: サムネイルプレビュー（またはアイコン）・タイトル・公開予定日・タグ・ステージバッジ・残り日数バッジ・メニューボタン
- 残り日数: 公開予定日まで7日以内は赤、14日以内はオレンジ、それ以外はグレー
- カードクリックで詳細モーダルを開く
- 「＋ 新規動画を追加」ボタン

### ② 動画詳細・編集モーダル

新規作成・既存編集を同一モーダルで行う。

**フィールド**
| フィールド | 型 | 備考 |
|----------|-----|------|
| タイトル（仮） | テキスト | 必須 |
| 公開予定日 | 日付 | 任意 |
| ステージ | セレクト | 5択 |
| メモ・備考 | テキストエリア | 任意 |
| サムネイル画像 | ファイルアップロード | Base64でlocalStorageに保存 |
| YouTube URL | URL | 公開後に記録、任意 |
| タグ | タグ入力（複数） | カンマ区切り or Enterで追加 |
| 作業時間 | タイマー（累積） | ステージ単位で計測・記録 |

**タイマー機能**
- 開始・停止ボタン
- 現在のステージに対して作業時間を累積記録
- localStorageに保存

**操作ボタン**
- 保存・キャンセル・削除（既存のみ）

### ③ カレンダービュー（`/calendar`）

- 月表示カレンダー
- 公開予定日が設定された動画のタイトルを該当日に表示
- ステージの色でタイル着色
- タイルクリックで詳細モーダルを開く
- 前月・翌月への移動ボタン

### ④ レポート画面（`/report`）

- **月別公開本数グラフ**: 過去12ヶ月の「公開」ステージに移行した動画本数を棒グラフで表示
- **現在のステージ別本数**: 円グラフまたは横棒グラフ
- **合計作業時間**: 全動画の累積タイマー時間の合計

## データモデル（localStorage）

```typescript
type Stage = '企画' | '台本' | '編集' | 'サムネイル' | '公開';

type WorkSession = {
  stage: Stage;
  startedAt: string; // ISO8601
  durationSeconds: number;
};

type Video = {
  id: string;           // UUID
  title: string;
  stage: Stage;
  publishDate?: string;    // ISO8601 date (YYYY-MM-DD)：公開予定日
  publishedAt?: string;    // ISO8601：「公開」ステージに移行した日時（月別グラフの集計に使用）
  memo?: string;
  thumbnailBase64?: string; // 保存前に1MB以下にリサイズ推奨（localStorage容量制限対策）
  youtubeUrl?: string;
  tags: string[];
  workSessions: WorkSession[];
  createdAt: string;    // ISO8601
  updatedAt: string;    // ISO8601
};
```

**localStorageキー**: `yt-progress-videos`（`Video[]` をJSON文字列で保存）

**注意**: localStorageの容量は通常5MB。サムネイル画像はアップロード時に自動リサイズ（長辺400px以下・JPEG圧縮）してBase64化することで容量を節約する。ステージを「公開」に変更したとき、`publishedAt` に現在日時を自動セットする。

## YouTube利用規約・セキュリティ

- YouTube Data API は一切使用しない
- 自動投稿・自動操作は行わない
- データはブラウザのlocalStorageにのみ保存し、外部サーバーへの送信は行わない
- アカウントBAN・利用規約違反のリスクはゼロ

---

## スコープ外（今回は含めない）

- ユーザー認証・ログイン機能
- チームでの共有・コラボレーション
- YouTube APIとの連携（アップロード・統計取得）
- プッシュ通知・リマインダー
- データのエクスポート・インポート
